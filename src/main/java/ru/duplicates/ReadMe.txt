Find duplicates.

Excample key for start, after deployed:
java -jar find.jar -d=c:/ -n=*.txt -t=mask -o=log.txt.
 
-d - directory. Directory for archiving. Example -d=c:\projects;
-n - file extension. Example -n=*.txt;
-t - mask. Use regular expressions;
-o - output loging. Application logging, log save to parrent path. Example -o=log.txt.
